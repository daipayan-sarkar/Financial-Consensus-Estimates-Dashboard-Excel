# Consensus Estimates Analysis Dashboard

![Excel Dashboard Preview](dashboard-preview.png)  

## 📋 Project Overview

This project simulates real-world **Consensus Estimates** workflows and demonstrates end-to-end financial data collation, broker estimate aggregation, consensus calculation, growth analysis, and professional reporting using Advanced Excel.

## 🎯 Key Objectives

- Replicate the process of collecting broker estimates for EPS and Revenue
- Calculate accurate consensus metrics (mean, median, high, low, # of estimates)
- Perform Year-over-Year (YoY) growth analysis
- Deliver insights through Pivot Tables and an Executive Dashboard

## 🛠️ Tech Stack
- **Excel**: Pivot Tables, Charts, Multi-sheet Reporting, Formatting

## 📊 Project Highlights

- **20 S&P 500 companies** across Technology, Financial Services, Healthcare, Consumer, and Energy sectors
- **1,000+ raw broker estimates** collated from 15+ leading firms (Goldman Sachs, J.P. Morgan, Morgan Stanley, Barclays, UBS, etc.)
- Consensus calculation for **EPS (USD)** and **Revenue (USD millions)** for FY2023 Actual to FY2026 Estimate
- Year-over-Year growth analysis and outlier detection
- Professional multi-sheet Excel output with:
  - Raw Estimates
  - Consensus Estimates
  - Sector-wise Pivot Tables
  - Growth Analysis
  - Executive Dashboard with Charts
## 📈 What I Learned / Skills Demonstrated

- Large-scale **financial data collation** and cleaning
- **Consensus methodology** (mean, median, high/low, standard deviation)
- Advanced **Pivot Tables** and dynamic Excel reporting
- **YoY growth analysis** for forward-looking insights
